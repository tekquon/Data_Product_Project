Estimating Male Body Fat Percentage with Shiny

Andrew Techmeier

Why Estimate Body Fat?

  1. It is important to monitor body fat to maintain and adjust ones health and lifestyle.
  2. Body fat is not an easily measured value. It typically requires special measurement devices.
  3. Using easily-measured data from the 'UsingR' 'fat' dataset, we can run a multivariate linear model to estimate body fat.

How It's Done

First fit a multivariate linear model to predict body fat percentage using the following predictors:

  • Age
  • BMI (using height and weight)
  • Neck, Chest, Abdomen, Hip, Thigh, Knee, Ankle, Bicep, Forearm, Wrist Circumference

The Model

library(UsingR)
data(fat)
simpleRegression <- lm(body.fat ~ age + BMI + neck + chest +  abdomen + hip + 
                               thigh + knee + ankle + bicep + forearm + wrist, 
                       data = fat)
summary(simpleRegression)
